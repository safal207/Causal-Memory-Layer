# CML Research Notes

This directory collects research-facing notes for Causal Memory Layer.

## Recommended reading order

1. [`CAUSAL_INVALIDITY_PATTERNS.md`](./CAUSAL_INVALIDITY_PATTERNS.md) -
   Start here to understand the current audit rule families R1-R4 and the
   causal-invalidity patterns they detect.
2. [`AGENTIC_WORKFLOW_CAUSAL_BOUNDARIES.md`](./AGENTIC_WORKFLOW_CAUSAL_BOUNDARIES.md) -
   Read next to see how the current taxonomy extends toward future agentic
   workflow boundary cases.
3. [`HIGH_VALUE_INTEGRATION_OPPORTUNITIES.md`](./HIGH_VALUE_INTEGRATION_OPPORTUNITIES.md) -
   Read this to understand the kinds of systems where CML-style causal audit
   could be relevant.
4. [`TECHNICAL_REPORT_OUTLINE.md`](./TECHNICAL_REPORT_OUTLINE.md) -
   Read this for the draft structure of a reviewer-facing technical report and
   grant narrative.
